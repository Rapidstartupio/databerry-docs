---
title: 'File Upload - Generate Link'
api: 'POST https://api.databerry.ai/datastores/file-upload/generate-link/{datastoreId}'
description: 'This endpoint allows to generate an upload link to upload a file to a datastore.'
authMethod: 'bearer'
---

### Path

<ParamField path="datastoreId" type="string" required>
  The id of the datastore where you want to upload a file.
</ParamField>

### Body

<ParamField body="type" type="string" required>
  Mime type of the file you want to upload.
</ParamField>

##### Supported mime types

- text/csv'
- text/plain'
- text/markdown'
- application/pdf'
- application/vnd.openxmlformats-officedocument.presentationml.presentation
- application/vnd.openxmlformats-officedocument.wordprocessingml.document
- application/vnd.openxmlformats-officedocument.spreadsheetml.sheet

### Response

<ResponseField name="id" type="string">
  Id of the upload link.
</ResponseField>
<ResponseField name="link" type="string">
  Generated link to upload a file.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.databerry.ai/datastores/file-upload/generate-link/<datastoreId>' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API_KEY>' \
--data-raw '{
    "type": "application/pdf"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": "60f9b9f0-0b1a-11ec-9a03-0242ac130003",
  "link": "https://example.com"
}
```

</ResponseExample>
