---
title: 'Authentication'
description: 'This demonstes how to authenticate a request to a private Datastore'
---

To access a private Agent or Datastore, you must provide a chaindesk API key in the Authorization header of your request.

```bash
'Authorization': 'Bearer <API_KEY>'
```

## API Keys

#### You can get your API key from your account settings page.
