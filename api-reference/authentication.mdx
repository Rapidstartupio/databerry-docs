---
title: 'Authentication'
description: 'This demonstes how to authenticate a request to a private Datastore'
---

To access a private Datastore, you must provide an API key in the Authorization header of your request.

```bash
'Authorization': 'Bearer <DATASTORE_API_KEY>'
```

## API Keys

#### You can get your API key from the Datastore settings page.
